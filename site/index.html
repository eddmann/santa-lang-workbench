Features

Everything you need for AoC

A thoughtfully designed IDE that streamlines your Advent of Code workflow with powerful features.

Run Solutions

Execute santa-lang code with real-time streaming output. Watch your solution progress as each part runs.

Multiple Reindeer

Download and manage 5 backend implementations. Switch between interpreters, VMs, and compilers instantly.

AoC Integration

Auto-detects aoc:// patterns and fetches puzzle descriptions and inputs automatically.

Comparative Testing

Run the same code on multiple reindeer simultaneously. Compare execution times with performance charts.

Code Formatting

Built-in formatting via Tinsel. Keep your santa-lang code clean and consistent with one keystroke.

Modern Editor

Monaco editor with santa-lang syntax highlighting, multiple tabs, and beautiful dark themes.

// Advent of Code 2023 - Day 15
let input = read("aoc://2023/15");
let hash = |s| {
s |> fold(0) { |acc, c|
(acc + ord(c)) * 17 % 256
}
};
Day 15: Lens Library

The newly-focused parabolic reflector dish is sending all of the collected light to a point on the side of yet another mountain...

The HASH algorithm is a way to turn any string of characters into a single number in the range 0 to 255.

To run the HASH algorithm on a string, start with a current value of 0.

Then, for each character:

  • Increase by ASCII code
  • Multiply by 17
  • Remainder after dividing by 256
